# workshop-03-04

A simple TODO management app.

## Features

- Add, edit, and delete TODO tasks
- Track task completion
- Simple, intuitive interface

## Getting Started

### Prerequisites

- [Python 3.x](https://www.python.org/downloads/) installed (if the app is in Python)
- Any other dependencies as listed in `requirements.txt` or `package.json`

### Installation

1. Clone the repository:
   ```sh
   git clone https://github.com/ksuvorau-g/workshop-03-04.git
   cd workshop-03-04
   ```
2. Install dependencies (if using Python):
   ```sh
   pip install -r requirements.txt
   ```

### Usage

Run the application:
```sh
python app.py
```
Or follow the instructions provided in the source code for other environments.
