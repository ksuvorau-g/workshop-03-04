# Agents Configuration

This file contains configuration and instructions for custom agents working on this project.

## Project Overview
A simple TODO management application with:
- **Backend**: Java 21 with Maven and Spring Boot
- **Frontend**: React

## Agent Guidelines

### Backend Agents
- Use Java 21 features where appropriate
- Follow Spring Boot best practices
- Maintain Maven project structure
- Ensure proper dependency management in pom.xml

### Frontend Agents
- Follow React best practices
- Use modern React patterns (hooks, functional components)
- Maintain clean component structure

### General Guidelines
- Write clean, maintainable code
- Follow existing code style conventions
- Add appropriate comments for complex logic
- Ensure proper error handling
